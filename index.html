SYNOPSIS

tom [OPTION] [FILE ...]

DESCRIPTION

tom is a wetware for Linux platforms, preferably Arch Linux and Gentoo but also works on Debian and Ubuntu. tom works best with Neovim and Vscode.

OPTIONS

-a, --ansible

read and write ansible playbooks and roles

-b, --bash

read and write bash scripts

-C

read and write C code. warning: option is not fully implemented

-html/-css

web development option is not fully implemented

-mk

build and configure mechanical keyboards

-p, --python

read and write python code
